# design-software
## Como rodar:
1. Criar venv
```python3 -m venv .venv```
2. Ativar venv
```source .venv/bin/activate```
3. Instalar bibliotecas
```pip install -r requirements.txt```
4. Rodar programa
```python3 main.py```
## inputs:
O projeto assume que os inputs de ID são inteiros, os valores de desconto são entre 0 e 1,
